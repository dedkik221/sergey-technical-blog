# Summary: Unicube

Unicube / AI automontage — личный проект в разработке для ускорения чернового монтажа разговорных видео. Подход: исходники или timeline -> транскрипт/SRT -> аудиоанализ -> selected phrases/scenario -> matching -> EditPlan -> FCPXML, markers или отчёт. Цель — не финальный автопилот, а контролируемый rough cut с review flags.
