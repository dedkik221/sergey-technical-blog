## Зачем я это делаю

Я много работаю с видео и контентом. Большая часть рутинных действий при монтаже коротких роликов — это техническая работа, которую можно автоматизировать. Цель проекта — сделать инструмент, который возьмёт на себя первый черновой слой и позволит фокусироваться на идеях и смыслах.

## Что уже работает

- Приём видео через Telegram-бот.
- Расшифровка речи через Whisper.
- Анализ аудио: поиск "крючков", пауз и важных моментов.
- Автоматический монтаж по заданным правилам.
- Генерация субтитров и оформление.
- Рендер в вертикальный формат 9:16.

## Технический стек

```text
Python, FastAPI, PostgreSQL, Redis
FFmpeg, faster-whisper, PyDub, MoviePy
Docker, Docker Compose
Работает на домашнем Linux-сервере без VPS
```

## Архитектура

Сервис полностью self-hosted. Пользователь отправляет видео в Telegram, бот принимает файл, обработчик разбирает звук и речь, слой правил собирает план ролика, рендеринг отдаёт готовое вертикальное видео. Файлы хранятся временно и удаляются автоматически.
